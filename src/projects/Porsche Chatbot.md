Developed a live data monitoring dashboard by integrating City of Toronto Bike Share API data into a Python application using Streamlit and Pandas. This enabled real-time tracking of bike and dock availability across stations.

Performed data validation and transformation by cleaning and structuring API responses for analysis and visualization, ensuring accurate availability metrics and reliable outputs.

Designed interactive data visualizations using geospatial mapping and color-coded indicators to highlight system performance and operational gaps.
